# NethServer 8 Mail Webhooks UI

This is the web user interface for the NethServer 8 Mail Webhooks module. The UI allows administrators to configure scheduled triggers and run one-time jobs to process emails from mailboxes and forward them to webhooks.

## Development

To develop the Mail Webhooks UI, please refer to [this section of the Developer manual](https://nethserver.github.io/ns8-core/ui/modules/#module-ui-development).

## Features

- Configure scheduled triggers for automated email processing
- Run one-time jobs for immediate processing
- Monitor processing status and statistics
- Manage webhook configurations with API keys
- Support for different payload formats (RAW email or JSON structured data)
